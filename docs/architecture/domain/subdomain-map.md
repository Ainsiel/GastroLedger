# Subdomain Map

| Subdomain | Type | Competitive importance | Complexity treatment |
|---|---|---|---|
| Recipe explosion and costing | Core | Explains theoretical consumption and margin | Rich domain rules, versions and tests |
| Lot-aware inventory and production | Core | Explains actual stock and variance | Ledger, aggregates and concurrency tests |
| Procurement | Supporting | Supplies real costs and lots | Workflow plus strong receiving rules |
| Store operations | Supporting | Supplies sales, cash and labor observations | Moderate workflow modeling |
| Franchise control and insights | Supporting | Compares branches and governs holds | Projections and policy calculations |
| Tenant/access/audit/jobs | Generic critical | Enables safe SaaS operation | Conservative, security-focused design |
